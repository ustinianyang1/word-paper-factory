# 快速入门指南

欢迎使用Word文档自动生成器！本指南将帮助您快速上手并生成您的第一个Word文档。

## 环境要求

- **Node.js**: 版本 14.0 或更高
- **npm**: 版本 6.0 或更高
- **浏览器**: 推荐使用最新版本的 Chrome、Firefox、Edge 或 Safari

## 安装步骤

### 1. 克隆或下载项目

```bash
git clone https://github.com/your-username/word-paper-factory.git
cd word-paper-factory
```

### 2. 安装依赖

```bash
npm install
```

## 基本操作

### 1. 启动开发服务器

```bash
npm run dev
```

服务器启动后，在浏览器中打开：
```
http://localhost:3000
```

### 2. 创建您的第一个文档

#### 步骤1：填写文档信息

在"文档编辑"标签页中：

1. **输入题目**：在"题目"输入框中输入您的论文题目（必填）
2. **添加个人信息**：点击"添加个人信息项"按钮，添加所需的个人信息
3. **填写正文**：在"正文内容"输入框中输入正文（必填）
4. **添加脚注**：在"脚注管理"区域添加脚注，在正文中使用 `[1]`、`[2]` 等格式标记脚注位置

#### 步骤2：自定义格式（可选）

切换到"格式设置"标签页，自定义各种格式，包括：
- 字体
- 字号
- 对齐方式
- 粗体

#### 步骤3：生成文档

点击右上角的"生成并下载文档"按钮，系统将自动生成Word文档并下载到您的浏览器默认下载目录。

## 核心功能说明

### 智能序号缩进

系统支持自动识别以下序号格式，并根据层级自动应用缩进：

- 一级：一、二、三、...
- 二级：（一）（二）（三）...
- 三级：1. 2. 3. ...
- 四级：（1）（2）（3）...
- 五级：①②③...

### 脚注支持

1. 在"脚注管理"区域点击"添加脚注"按钮
2. 在正文中使用 `[数字]` 格式标记脚注位置
3. 在脚注输入框中填写脚注内容

## 常见问题

### Q1: 安装依赖时出现错误？

**A**: 请尝试清除npm缓存并重新安装：
```bash
npm cache clean --force
rm -rf node_modules
npm install
```

### Q2: 为什么预览中某些部分不显示？

**A**: 当某个部分（如摘要、关键词）未输入内容时，系统会自动隐藏该部分，以提升预览效果。
