# Word文档自动生成器

一个基于React + TypeScript的Word文档自动生成工具，支持自定义格式设置，一键生成符合学术规范的Word文档。

## 功能特性

- 📝 **文档信息管理**：输入题目、自定义个人信息项
- 📄 **内容编辑**：编辑摘要、关键词、引言、正文、结论、参考文献
- 📚 **脚注支持**：支持添加和管理脚注
- 🎨 **格式自定义**：可自定义所有元素的字体、字号、对齐方式、行距等
- 🔢 **智能序号缩进**：自动识别正文序号，根据序号层级应用适当的缩进样式
- 👁️ **实时预览**：实时预览文档效果，自动隐藏空内容部分
- 💾 **配置保存**：支持导出和导入格式配置
- 📥 **一键生成**：生成符合命名规则的Word文档

## 使用说明

### 1. 填写文档信息

在"文档编辑"标签页中，填写以下信息：
- **题目**：输入论文题目（必填）
- **个人信息**：点击"添加个人信息项"按钮，自定义所需的个人信息（可选）
- **摘要**：输入论文摘要内容（可选）
- **关键词**：输入关键词，用分号分隔（可选）
- **正文内容**：输入正文，支持多行输入和脚注标记（可选）
- **结论**：输入结论内容（可选）
- **参考文献**：输入参考文献（可选）

### 2. 自定义格式（可选）

在"格式设置"标签页中，可以自定义各种格式，包括：
- 题目格式、个人信息格式
- 摘要、关键词、引言格式
- 正文、结论、参考文献格式
- 脚注格式
- 序号缩进设置

#### 行间距设置

1. 进入"格式设置"标签页
2. 展开需要设置行间距的部分（如"引言内容格式"、"正文格式"等）
3. 在"行间距"下拉菜单中选择所需的行间距值（1.0倍、1.15倍、1.5倍、2.0倍）
4. 实时预览效果会立即更新

#### 智能序号缩进设置

1. 进入"格式设置"标签页
2. 展开"序号缩进设置"部分
3. 为每个级别的序号设置合适的缩进值（0-20em，步长0.5em）
4. 实时预览效果会立即更新

支持的序号格式：
- 一级：一、二、三、...
- 二级：（一）（二）（三）...
- 三级：1. 2. 3. ...
- 四级：（1）（2）（3）...
- 五级：①②③...

### 3. 生成文档

点击右上角的"生成并下载文档"按钮，系统将自动生成Word文档并下载。
